msdn 解释如下：

“协变”是指能够使用与原始指定的派生类型相比，派生程度更大的类型。

“逆变”则是指能够使用派生程度更小的类型。

“协变”->”和谐的变”->”很自然的变化”->string->object :协变。(儿子变成父亲)

“逆变”->”逆常的变”->”不正常的变化”->object->string 逆变。（父亲变成儿子）


上面是个人对协变和逆变的理解，比起记住那些派生，类型，原始指定,更大，更小之类的词语，个人认为要容易点。
